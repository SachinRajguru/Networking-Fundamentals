## Introduction

Networking Fundamentals is the foundation of how the internet works. Just like understanding how letters are delivered through a postal system helps you grasp the concept of mailing, understanding networking helps you comprehend how data travels across the internet.

In simple terms, when you open a website, send an email, or stream a video, data travels through a complex network of devices and protocols. This guide breaks down those concepts into easy-to-understand pieces.

Whether you are building applications, managing cloud infrastructure, or simply curious about how the internet works, this guide will provide you with the essential knowledge needed to understand and work with networked systems confidently.

---

## Purpose

- This guide covers basic networking concepts anyone can understand
- Uses simple examples
- Essential for roles like **DevOps**, **Data Engineers**, **Full-Stack Engineers**, or anyone working with infrastructure or applications

---

## Key Topics Covered

- IP Addresses
- Subnets
- CIDR (Classless Inter-Domain Routing)
- Ports
- OSI Model (including prerequisites like DNS resolution and TCP handshake)

## Why Learn This?

Networking fundamentals help you understand:

- How devices connect
- How data flows across networks
- How to troubleshoot issues in your organizational setup or application workflows
- How to design secure infrastructure
